Open Source Project

FAST

v3.00.03

File Download and Signal Processing Toolkit for GNSS

一个基于 Python 开发的高效多模 GNSS 数据预处理框架。集成了数据下载、质量分析、SPP 定位及测站筛选四大核心功能，支持 GUI 与 命令行操作。

View on GitHub
Star Fork GPL-3.0

Core Modules & Features

Data Download

支持 IGS/WHU/COD 等 13+ 种数据源。采用多线程并发下载，自动解压 (.gz, .Z) 及格式转换 (crx2rnx)。

Quality Analysis

提供周跳探测 (MW/GF组合)、多路径效应分析 (Multipath)、信噪比 (CNR) 及电离层延迟率 (IOD) 可视化。

SPP Positioning

基于双频无电离层组合模型。支持 GPS/BDS/Galileo 多系统标准单点定位，自动解算坐标与接收机钟差。

Station Selection

基于经纬度范围、天线类型及系统可见性的 IGS 测站可视化筛选工具，支持自动抽稀。

Research & Engineering

微厘空间一号组网卫星工程导航增强系统

2022.01 - PRESENT

"设计并实现了自主可控的低轨卫星精密定轨与星历生成系统，成功保障2025年1月组网星发射任务。"

  • 精密定轨 (POD) 算法攻关： 针对低轨卫星动力学环境，优化了非保守力模型，实现了厘米级的事后精密定轨精度。
  • 星历生成全流程研发： 构建了从原始观测数据处理、轨道积分到广播星历参数拟合的完整业务链路。
  • 增强导航定位验证： 基于 LEO 实测数据，评估了星地链路质量，并验证了低轨增强对 PPP 收敛时间的改善效果。
